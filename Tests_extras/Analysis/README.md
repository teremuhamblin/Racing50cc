# Analysis — Scripts d’Analyse Automatique

## 🎯 Objectif
Automatiser l’analyse des données issues :
- des tests dynamiques
- de la télémétrie
- des performances enregistrées

L’objectif est d’obtenir :
- des courbes automatiques
- des statistiques
- des comparaisons entre configurations
- des rapports d’analyse

---

# 📊 Types d’analyses

## Analyse accélération
- Temps 0–50 m
- Temps 0–100 m
- Reprise 20–60 km/h
- Courbe vitesse/temps

## Analyse moteur
- Régime max
- Stabilité du régime
- Température moyenne
- AFR (si capteur)

## Analyse transmission
- Plage variateur
- Régime embrayage
- Régime stabilisé
