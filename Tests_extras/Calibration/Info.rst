Calibration — Informations
==========================

Objectif
--------
Définir des procédures de calibration reproductibles pour :
- capteurs (température, AFR, vitesse)
- moteur (ralenti, avance, carburation)
- transmission (tension chaîne, alignement)

Organisation
------------
- procedures/ : protocoles de calibration
- reference/ : valeurs de référence
- tools/ : outils ou gabarits nécessaires
