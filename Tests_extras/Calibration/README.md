# Calibration — Procédures de Calibration Capteurs

## 🎯 Objectif
Garantir la précision des mesures :
- RPM
- CHT
- EGT
- GPS
- Accéléromètre

---

# 🔧 Procédures de calibration

## RPM
- Vérification avec compte-tours externe
- Ajustement du facteur de conversion

## CHT
- Test eau bouillante (100°C)
- Test eau glacée (0°C)

## EGT
- Test flamme contrôlée
- Vérification montée progressive

## GPS
- Test en ligne droite
- Comparaison avec smartphone

## Accéléromètre
- Calibration 6 axes
- Test vibrations moteur
