# Tests_extras/

Dossier dédié aux tests avancés, complémentaires aux tests dynamiques standards.  
Il regroupe trois modules spécialisés : **analysis**, **calibration** et **export**.

## 📁 Structure

- **analysis/**  
  Analyse approfondie des données (RPM, vitesse, AFR, T°C, charge).  
  Scripts, méthodes et rapports d’interprétation.

- **calibration/**  
  Procédures de calibration moteur et capteurs.  
  Réglages précis, protocoles reproductibles, valeurs de référence.

- **export/**  
  Formats d’export optimisés (CSV, JSON, RST).  
  Préparation des données pour exploitation externe.

## 🎯 Objectif du dossier
Fournir un espace dédié aux tests avancés permettant :
- d’améliorer la précision des mesures  
- d’affiner les réglages moteur  
- d’optimiser les analyses  
- de faciliter le partage et l’exploitation des données
