# Tuning — Réglages Avancés Scooter 50cc Racing

## 🎯 Objectif
Centraliser tous les réglages avancés permettant d’optimiser le comportement du scooter selon :
- La piste
- La météo
- Le type de compétition (run / circuit)
- Le style de pilotage

---

# 🔥 Réglages Avancés Moteur

## Diagramme d’admission
- Ajustement clapets
- Angle d’ouverture
- Débit d’air optimisé

## Préparation haut moteur
- Squish : 0.55–0.65 mm
- Volume de chambre : 5.5–6.2 cc
- Polissage échappement (miroir)

---

# ⚙️ Réglages Avancés Transmission

## Variateur
- Galets asymétriques
- Plage optimisée pour 13 500–14 200 tr/min

## Embrayage
- Ressorts ultra durs pour départ run
- Ressorts medium pour circuit

## Correcteur de couple
- Angle 38° recommandé pour 50cc racing

---

# 🎛️ Réglages Avancés Carburation

## Température idéale
- 55–65°C

## Ajustements selon météo
- Air sec : gicleur +2
- Air humide : gicleur -2
- Température basse : aiguille +1 cran

---

# 🧪 Méthodes de réglage
- Lecture bougie
- Analyse sonore
- Test pleine charge
- Test reprise 20–60 km/h

---
