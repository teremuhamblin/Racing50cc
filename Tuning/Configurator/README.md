# Configurator — Racing50cc

Ce dossier contient le **Configurateur Moteur Racing50cc**, un outil interactif permettant de définir une configuration moteur en fonction :

- du cylindre  
- de l’échappement  
- du carburateur  
- du type d’utilisation (street, sprint, piste)

Le configurateur génère automatiquement un **profil moteur** basé sur les choix sélectionnés.

---

## 📁 Contenu

- `index.html` — interface du configurateur  
- `configurator.js` — logique de calcul du profil moteur  
- `configurator.css` — style visuel (dark racing)  

---

## 🎯 Objectif

Fournir un outil simple et rapide pour :

- comparer plusieurs configurations  
- estimer le comportement moteur  
- orienter les réglages selon l’usage  
- harmoniser les choix de pièces (Stage6, Polini, Girardo)

---

## 🧪 Fonctionnement

1. Choisir les éléments de configuration (cylindre, carbu, échappement).  
2. Sélectionner le type d’utilisation.  
3. Le configurateur calcule automatiquement :  
   - un **profil moteur**  
   - une **orientation sponsor** (Stage6 / Polini / Girardo)  
   - une **recommandation d’usage**  

---
