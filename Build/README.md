# Build — Pièces & Références du Projet Racing50cc

## 🎯 Objectif
Le dossier **build** regroupe l’ensemble des pièces utilisées pour la préparation du **Piaggio Zip SP 50cc Racing**, ainsi que leurs références officielles, leurs caractéristiques techniques et leurs origines de fabrication.

Ce dossier sert de base pour :
- sélectionner les pièces,
- vérifier leur compatibilité,
- documenter les valeurs techniques,
- assurer la reproductibilité du projet.

---

# 📂 Contenu du dossier

## `pieces_stage6.md`
Catalogue détaillé des pièces Stage6 utilisées dans le projet :
- Cylindre, vilebrequin, admission, allumage, échappement
- Transmission complète
- Refroidissement
- Freinage
- Pneus
- Matériaux, usines, fabrication
- Valeurs techniques précises

## `references_stage6.md`
Tableaux de références officielles Stage6 :
- Références exactes
- Valeurs techniques
- Origine de fabrication
- Usines de production
- Compatibilité avec le Piaggio Zip SP

# 🏁 Marque principale utilisée : Stage6 France

Stage6 est une marque spécialisée dans les pièces racing pour scooters 50cc :
- Conception : Allemagne (SIP Group)
- Production : Europe (Allemagne, Italie) + Asie CNC
- Distribution France : Stage6 France
- Réputation : hautes performances, pièces fiables, qualité compétition

---

# 🔧 Utilisation dans le projet
Les pièces listées dans ce dossier sont utilisées pour :
- la configuration moteur 50cc racing,
- la transmission haute performance,
- le refroidissement optimisé,
- le châssis et le freinage,
- les tests dynamiques et simulations.
