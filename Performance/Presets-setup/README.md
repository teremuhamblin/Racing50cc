# Setup Presets — Configurations Prêtes à l’Emploi

## 🎯 Objectif
Proposer des configurations complètes selon :
- le type de piste
- la météo
- le style de pilotage
- la catégorie (run / circuit)

---

# 🏁 Presets disponibles

## Run (départ arrêté)
- Galets légers
- Ressorts embrayage durs
- Courbe allumage agressive
- Pression pneus basse

## Circuit
- Galets moyens
- Ressorts medium
- Courbe allumage progressive
- Pression pneus équilibrée

## Temps froid
- Gicleur +2
- Aiguille +1 cran

## Temps chaud
- Gicleur -2
- Allumage légèrement retardé
