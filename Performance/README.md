# Performance — Analyses, Courbes & Logs

## 🎯 Objectif
Analyser les performances du scooter pour :
- Comprendre son comportement réel
- Optimiser les réglages
- Comparer les configurations
- Suivre l’évolution du projet

---

# 📈 Courbes de Performance

## Courbes recommandées
- Courbe de puissance (rpm → hp)
- Courbe de couple (rpm → Nm)
- Courbe AFR (air/fuel ratio)
- Courbe température moteur
- Courbe accélération (temps → vitesse)

## Formats
- PNG / SVG pour les visuels
- CSV / JSON pour les données brutes
