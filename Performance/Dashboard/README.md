# Dashboard Performance — Racing50cc

Ce dossier contient le **Dashboard Performance Racing50cc**, un outil visuel permettant d’afficher :

- les courbes de puissance  
- les données d’accélération  
- les résultats de tests dynamiques  
- les comparaisons entre configurations  

Le dashboard est conçu pour analyser rapidement les performances du Piaggio Zip SP dans le cadre du projet Racing50cc.

---

## 📁 Contenu

- `index.html` — interface du dashboard  
- `dashboard.js` — logique d’affichage et données  
- `dashboard.css` — style visuel (dark racing)  

---

## 🎯 Objectif

Fournir une interface simple et efficace pour :

- visualiser les performances moteur  
- afficher les courbes (via Chart.js)  
- suivre l’évolution des configurations  
- comparer les résultats entre versions (v1.x → v3.x)  

---

## 🚀 Utilisation

Ouvrir simplement :
index.html

Le dashboard se charge automatiquement avec les données intégrées dans `dashboard.js`.

---

## 🧪 Fonctionnement

Le dashboard affiche :

- une **courbe de puissance**  
- un **résumé d’accélération 0–50 km/h**  
- des données dynamiques modifiables  

Les données peuvent être remplacées par :

- des logs réels  
- des mesures dynamiques  
- des résultats de simulateurs  

---

## 🛠 Technologies

- HTML  
- CSS (dark racing theme)  
- JavaScript  
- Chart.js (graphiques dynamiques)

---

## 📌 Notes

Le dashboard est conçu pour être extensible.  
Tu peux y ajouter :

- plusieurs courbes  
- des comparaisons multi-configurations  
- des données issues du simulateur transmission/accélération  
- des exports CSV ou JSON
