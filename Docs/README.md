# Projet Racing50cc
>Documentation Officielle

### 🎯 Objectif
>Créer un scooter 50cc entièrement optimisé pour la compétition (run ou circuit), basé sur un **Piaggio Zip SP**, reconnu pour sa fiabilité, son châssis léger et son moteur performant.

### 📦 Contenu de la documentation
- Architecture moteur, transmission et châssis
- Préparation complète 50cc racing
- Réglages carb
