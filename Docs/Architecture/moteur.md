# Architecture Moteur — Piaggio Hyper2 LC

## Vue générale
Le moteur Piaggio Hyper2 LC est un 2T liquide réputé pour :
- Sa robustesse
- Sa plage de régime exploitable
- Sa compatibilité avec les pièces racing
- Sa facilité de préparation

## Caractéristiques techniques
- Cylindrée : 49.9cc
- Alésage : 40mm
- Course : 39.3mm
- Admission lamelles
- Allumage externe (remplaçable par interne rotor)
- Refroidissement liquide

## Zones critiques à renforcer
- Roulements C4 haute vitesse
- Joints spy renforcés
- Alignement carters
- Renfort de jupe cylindre
- Étanchéité admission

## Couples de serrage
- Culasse : 12 Nm  
- Cylindre : 14 Nm  
- Carter : 10 Nm  
- Volant : 45 Nm  

## Schéma d’explosion (description)
- Haut moteur : culasse → cylindre → piston → axe → cage à aiguilles  
- Bas moteur : vilebrequin → roulements → joints → carters
