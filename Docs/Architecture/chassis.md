# Architecture Châssis — Piaggio Zip SP

## Caractéristiques
- Châssis tubulaire léger
- Empattement court
- Très maniable
- Idéal pour run et circuit

## Optimisations recommandées
- Allègement (carénages, béquille, support)
- Amortisseur racing
- Frein avant oversize
- Pneus slicks PMT
- Guidon bas pour meilleure position
