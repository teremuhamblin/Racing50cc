# Réglages Carburation

## Gicleurs
- Principal : 90–105 selon échappement
- Ralenti : 45–50

## Aiguille
- Clip milieu (ajuster selon réponse moteur)

## Boisseau
- 30–40 selon comportement bas régime

## Lecture bougie
- Marron clair = parfait
- Blanc = trop pauvre
- Noir = trop riche

## Symptômes
- Ratés haut régime = trop pauvre
- Engorgement = trop riche
