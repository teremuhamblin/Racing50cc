# Réglages Transmission

## Objectif
Maintenir le moteur dans sa plage de puissance.

## Galets
- Plus légers = plus de régime
- Plus lourds = plus de pointe

## Ressorts embrayage
- Durs pour départ agressif
- Moyens pour circuit

## Ressort de poussée
- +30% recommandé

## Tests
- Plage variateur
- Régime max
- Temps d’accélération
