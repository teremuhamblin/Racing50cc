# Préparation Moteur 50cc Racing

## Objectif
Maximiser les performances tout en restant dans la catégorie 50cc.

## Configuration recommandée
- Cylindre : Stage6 Racing MK2 50cc
- Vilebrequin : Stage6 HPC
- Roulements : C4 haute vitesse
- Carburateur : 21mm PHBG
- Admission : VForce Stage6
- Allumage : PVL ou MVT Premium
- Échappement : Stage6 R1200

## Préparation carters
- Alignement admission
- Polissage échappement
- Suppression défauts de fonderie
- Ajustement transfert bas moteur

## Refroidissement
- Radiateur gros volume
- Pompe à eau électrique
- Durites silicone

## Étanchéité
- Joints neufs
- Pâte haute température
