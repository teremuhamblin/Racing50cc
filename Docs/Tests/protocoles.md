# Protocoles de Tests

## Test d’accélération
- 0–50 m
- 0–100 m
- Mesure GPS ou chronomètre

## Test de pointe
- Ligne droite 200 m
- Vent faible
- Position constante

## Test carburation
- Lecture bougie
- Réaction poignée
- Test pleine charge

## Test transmission
- Plage variateur
- Régime max
- Temps de montée en régime
