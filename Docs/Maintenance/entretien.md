# Maintenance & Entretien

## Après chaque sortie
- Nettoyage filtre
- Vérification bougie
- Tension courroie
- Contrôle serrages

## Toutes les 10h
- Changement piston
- Nettoyage carburateur
- Vérification allumage

## Toutes les 20h
- Vérification vilebrequin
- Contrôle roulements
- Contrôle transmission

## Long terme
- Remplacement joints
- Révision complète moteur
