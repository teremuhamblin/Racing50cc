# Diagnostics — Détection Automatique de Problèmes Moteur

## 🎯 Objectif
Détecter automatiquement les anomalies moteur à partir :
- des données de télémétrie
- des tests dynamiques
- des analyses automatiques

---

# 🛠️ Types de diagnostics

## Surchauffe
- CHT > 70°C
- EGT > 700°C

## Mauvaise carburation
- AFR trop pauvre ou trop riche
- Bougie blanche/noire

## Transmission
- Variateur qui ne monte pas en plage
- Embrayage qui patine

## Allumage
- Coupures haut régime
- Avance incorrecte
