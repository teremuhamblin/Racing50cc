# Racing50cc
- Piaggio Zip SP

### 🏁 Présentation
Projet complet de préparation d’un **Piaggio Zip SP 50cc racing**, optimisé pour la compétition (run ou circuit).

### 📚 Documentation
Toute la documentation est disponible dans le dossier `/docs`.

### 🧱 Structure du projet
- `/docs` — Documentation complète  
- `/build` — Liste des pièces + références  
- `/project` — Workflow, architecture, planning  
- `/assets` — Logos, schémas, visuels  

### 🛠️ Objectifs techniques
- Maximiser les performances moteur 50cc
- Optimiser la transmission
- Alléger et rigidifier le châssis
- Obtenir un scooter fiable et performant
