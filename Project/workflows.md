# Project — Architecture & Workflow du Projet Scooter 50cc Racing

## 🎯 Objectif
Structurer le projet de préparation du Piaggio Zip SP 50cc racing.

---

# 🧱 Architecture du Projet

## 1. Moteur
- Préparation bas moteur
- Alignement carters
- Montage vilebrequin
- Haut moteur racing
- Admission + carburateur
- Allumage interne rotor

## 2. Transmission
- Variateur racing
- Embrayage renforcé
- Rapport secondaire
- Tests plage variateur

## 3. Châssis
- Allègement
- Suspension
- Freinage
- Pneus slicks
